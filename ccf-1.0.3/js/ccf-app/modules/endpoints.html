AuthnIdentity: EmptyAuthnIdentity | UserCertAuthnIdentity | MemberCertAuthnIdentity | UserSignatureAuthnIdentity | MemberSignatureAuthnIdentity | JwtAuthnIdentity

The type of an endpoint function. This type definition is typically not directly used. Endpoint functions are usually defined as follows:
